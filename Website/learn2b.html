Synopsis Skip & Start the quiz

  1. File Management:
    pwd: Display the current working directory.
    mkdir: Create a new directory.
    rmdir: Delete a directory.
    ls: List the contents of a directory.
    cd: Change the current directory.
  2. File Content Manipulation:
    touch: Create empty files.
    cat: Display, create, or copy file content.
    rm: Remove a file.
    cp: Copy a file or directory.
    mv: Move or rename a file or directory.
  3. User Management:
    su: Switch user.
    id: Display user and group IDs.
    useradd: Add a user.
    passwd: Create or change a user's password.
    groupadd: Create a user group.

  4. Filtering Commands:
    grep: Search for specific content within a file.
    cut: Select specific columns from a file.
    sed: Edit files using regular expressions.
    sort: Sort files in alphabetical order.
    uniq: Display unique lines in a sorted file.
  5. Utility Commands:
    find: Search for files within a directory.
    locate: Locate files by their names.
    date: Display date and time information.
    sleep: Pause the execution for a specified time.
    time: Measure the execution time of a command.

Instructions

  1. Ensure you have a stable internet connection.
  2. Click on the "Start Quiz" button below to begin.
  3. You will be presented with a series of questions.
  4. Select the most appropriate answer for each question.
  5. You can only choose one answer for each question.
  6. Keep a watch at time try to solve all questions with in 5 minitues.
  7. Once you have answered for each questions, click the "Save" button followed by "Final Submit" button.
  8. You will receive your quiz score and feedback on the results page.
