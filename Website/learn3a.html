  1. MySQL INNER JOIN: The INNER JOIN keyword selects records that have matching values in both tables.
  2. MySQL RIGHT JOIN: The RIGHT JOIN keyword returns all records from the right table, and the matching records from the left table.
  3. MySQL LEFT JOIN: The LEFT JOIN keyword returns all records from the left table, and the matching records from the right table.
  4. MySQL CROSS JOIN: The CROSS JOIN keyword returns all records from both tables
  5. MySQL Self Join: A self join is a regular join, but the table is joined with itself.
