Synopsis Skip & Start the quiz

  1. ip -
    Manipulating routing to assigning and configuring network parameters
  2. traceroute - Identify the route taken by packets to reach the host
  3. tracepath - Gets maximum transmission unit while tracing the path to the network host

  4. ping - Often used to check the connectivity between the host and the server
  5. ss - Gets details about network sockets
  6. dig - Gives all the necessary information about the DNS name server
  7. host - Prints IP address of a specific domain and viscera
  8. hostname - Mostly used to print and change the hostname
  9. curl - Transfers data over the network by supporting various protocols
  10. mtr - A combination of ping and traceroute is used to diagnose the network
  11. whois - Gets info about registered domains, IP addresses, name servers, and more
  12. ifplugstatus - Detects the link status of a local Ethernet device
  13. iftop - Monitors stats related to bandwidth
  14. tcpdump - Packet sniffing and analyzing utility used to capture, analyze and filter network traffic
  15. ethtool - Allows users to configure Ethernet devices
  16. nmcli - Troubleshooting utility for network connections
  17. nmap - Primarily used to audit network security
  18. bmon - An open-source utility to monitor real-time bandwidth
  19. firewalld - CLI tool to configure rules of Firewall
  20. iperf - Utility to measure network performance and tuning
  21. speedtest-cli - CLI utility of speedtest.net to check internet speeds
  22. vnstat - Mostly used to monitor network traffic and bandwidth consumption
